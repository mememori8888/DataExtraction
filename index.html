データ抽出プログラム

APIを活用したデータ抽出プログラムの紹介です。ご利用にあたり、以下の手順に従って設定を行ってください。

1 APIキーの設定

データ抽出機能を利用するには、AIのAPIキーをご準備ください。 まずは、`api_key.json` ファイルを作成し、ご準備いただいたAPIキーを以下の形式で記述してください。 

{
  "api_key": "あなたのAPIキーをここに貼り付けてください"
}

※ APIキーは慎重に取り扱い、公開されないようにしてください。

2 settings.jsonの設定

次に、`settings.json` ファイルを作成し、GitHub Actionsの設定を行います。 以下にサンプルを示します。各項目はプロジェクトに応じて設定してください。 

[
  {
    "task_name": "funeral_search",
    "query": "斎場||告別式||葬儀場||葬儀サービス||家族葬||一日葬||直葬||ホール||葬儀",
    "address_csv_path": "address.csv",
    "facility_file": "funeral.csv",
    "review_file": "funeral_review.csv",
    "update_facility_path": "funeral_add_data.csv",
    "update_review_path": "funeral_add_review.csv"
  }
]
  • ・`query`: 業種を指定するためのキーワードを設定します。
  • ・`address_csv_path`: 地域とリクエスト数を指定するCSVファイルへのパスです。
  • ・`facility_file`: 施設データを格納するファイルです。
  • ・`review_file`: レビューデータを格納するファイルです。
  • ・`update_facility_path`と`update_review_path`: 照合後に新しく増えたデータのみを記載するファイルです。

3 `main3_api_key_json.py` の起動

上記の設定が完了したら、`main3_api_key_json.py` を起動してください。
GitHub Actionsの場合は、**Actions**タブをクリックして、**action name**を選択して起動します。
更新されたファイルはGitHubからダウンロードしてご利用いただけます。
